Этот проект создается как треккер личных расходов. Основной функционал:

  1. Учет денежных средств с возможностью разбивки по разным счетам, картам.

  2. Учет понесенных затрат. 

  3. Внесение будущих затрат с построением графика остатка денежных средств с учетом предпологаемого движения денежных средств.

  4. Ежемесячный отчет о понесенных затратах с учетом контрагентов и категорий.
